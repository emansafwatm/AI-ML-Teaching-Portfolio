1. Data cleaning
2. Exploratory data analysis
3. Visualization
4. Descriptive statistics
5. Correlation and association
6. Grouping, aggregation, and pivot tables
7. Statistical hypothesis testing

Students should be able to clean, explore, visualize, summarize, test, and communicate findings from structured datasets.
